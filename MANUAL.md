# 喵喵和声 Pro Max - 使用说明

欢迎使用喵喵和声，你的智能吉他/和声助手！

## 核心理念：和声色彩理论 (Harmony Color)

本应用的核心是“和声色彩”理论的简化模型。我们将每个和弦都视为一个向量，它有两个关键属性：

1.  **张力 (Tension)**: 向量的**半径 (r)** 表示。张力越低（半径越小），和弦听起来越“解决”、“稳定”；张力越高（半径越大），和弦听起来越“悬置”、“不稳定”，需要被解决。例如，主和弦（Tonic）的张力最低，而属和弦（Dominant）和导和弦（Leading-tone）的张力则很高。

2.  **色彩 (Color)**: 向量的**角度 (theta)** 表示。它大致对应于和弦在五度圈上的位置。相似色彩的和弦通常在功能和听感上都比较接近。

通过将和弦进行可视化，你可以更直观地理解和弦进行(progression)的“流动”趋势：一个高张力的和弦如何解决到一个低张力的和弦，或者如何在色彩相近的和弦之间平滑过渡。

## 各个模块说明

### 1. 和弦选择 (Chord Selector)

这是你构建和弦的核心区域。

-   **根音 (Root)**: 选择和弦的根音，从 C 到 B。
-   **类型 (Type)**: 选择和弦的类型。我们提供了从基础的大三、小三和弦到复杂的九和弦、挂留和弦等等。每个和弦类型都有其中文全称，方便你理解。
-   **低音 (Bass)**: 选择和弦的转位低音。默认“无”表示为原位和弦。
-   **试听/添加**: 你可以随时试听当前选择的和弦，或者将其添加到下方的“和弦序列”中。
-   **指板点选模式**: 点击“切换到指板点选”后，你可以在下方的吉他指板上自由选择音符，应用会自动识别你选择的和弦。

### 2. 我的和弦序列 (My Chord Sequence)

这里是你创作的“乐谱”。

-   你添加的和弦会以卡片的形式在这里列出。
-   每个卡片上都会显示和弦名称、以及它在当前设定的调性（Key）下的**级数 (Degree)**。例如，在 C 大调中，G7 和弦就是 V7 级。
-   你可以拖动卡片来改变和弦的顺序。
-   你可以点击“播放”按钮，以设定的BPM和节奏型来播放整个序列。
-   **插入经过和弦**: 在两个和弦之间，你可以点击 `+` 按钮，AI 会为你推荐一个合适的经过和弦（通常是下一个和弦的属和弦，即 V/Next），使进行更流畅。

### 3. AI 灵感作曲猫 (AI Composer Cat)

这是本应用的“最强大脑”。

-   **API Key**: 你需要在这里输入你的大模型 API Key 才能使用 AI 功能。
-   **AI 模型提供商**: 你可以选择使用不同的 AI 模型，例如 Google Gemini, Anthropic Claude, OpenAI GPT 等。如果你选择 OpenRouter，还可以自定义模型名称。
-   **目标风格/氛围**: 告诉 AI 你想要的音乐感觉，例如“温暖治愈”、“日系摇滚”或“意外转调”。
-   **给 AI 的悄悄话**: 你可以在这里输入更具体的、个性化的要求，AI 会尽力理解。
-   **获取灵感**: 点击后，AI 会根据你当前的和弦序列、目标风格和自定义要求，为你推荐 3 个可能的下一个和弦，并给出推荐理由。你可以直接点击推荐的和弦，将其应用到上方的“和弦选择区”。

### 4. 深度分析 (Analysis)

在“深度分析”标签页，你可以看到更多关于和声色彩的可视化信息。

-   **向量与色彩分析**:
    -   **五度圈**: 这是一个基于五度圈的向量可视化圆盘。你可以看到当前选择的和弦以及整个序列中所有和弦的“色彩”和“张力”分布。
    -   **数据**: 当前和弦的张力、色彩角度、音符数等精确信息。
-   **图表分析**:
    -   **张力变化图**: 你的和弦序列的张力是如何随时间变化的。一个好的进行通常有起伏的张力曲线。
    -   **色彩变化图**: 你的和弦序列的色彩（在五度圈上的位置）是如何移动的。

### 5. 吉他指板 (Fretboard)

位于应用底部的吉他指板会实时显示当前选中和弦的按法。

-   它会标出每个音符的名称以及是否是根音。
-   在“指板点选”模式下，这里就是你的创作画布。
